# Web Tailwind CSS - Proyecto Educativo

## 📚 Descripción

Proyecto educativo basado en el curso **[Curso de Tailwind CSS desde Cero hasta Profesional](https://www.udemy.com/course/curso-de-tailwind-css-desde-cero-hasta-profesional)** de Udemy.

La intención de este proyecto es **transformar un proyecto frontend previamente desarrollado** (usando CSS tradicional) para implementar **Tailwind CSS** como framework de utilidades.

## 🔄 Diferencia Importante

- **Curso original**: Utiliza **Tailwind CSS 2**
- **Este proyecto**: Implementación con **Tailwind CSS 4** + **Vite**

Se busca explorar y adaptar los conceptos del curso a las versiones más recientes de las tecnologías, aprovechando las mejoras y características nuevas.

## 🛠️ Stack Tecnológico

- **Tailwind CSS**: ^4.3.1
- **Vite**: ^8.1.0
- **TypeScript**: ~6.0.2
- **@tailwindcss/vite**: ^4.3.1

## 📋 Estructura del Proyecto

```
web-tailwindcss/
├── public/
│   ├── css/
│   │   ├── styles.css (estilos personalizados)
│   │   ├── responsive.css (media queries)
│   │   └── fonts/
│   ├── img/
├── src/
│   └── style.css (configuración de Tailwind)
├── index.html
├── vite.config.ts
├── tsconfig.json
└── package.json
```

## 🚀 Instalación y Uso

```bash
# Instalar dependencias
npm install

# Ejecutar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Vista previa de la compilación
npm run preview
```

## 📖 Objetivo de Aprendizaje

Este proyecto sirve como práctica para:
- Dominar las utilidades de Tailwind CSS
- Refactorizar CSS tradicional a utilidades de Tailwind
- Configurar Tailwind CSS 4 con Vite
- Aplicar responsive design con clases de Tailwind
- Optimizar CSS con Tailwind en producción
